# jira-integration

## Сбор/изменение информации в JIRA/Insight из различных сервисов:
### VMWare vCenter (RTCloud)
Стоимость каждоый вм, её состояние, в каком виртуальном ЦОДе расположена, использовано/выделено RAM, CPU. Стоимость каждого ЦОДа и потребляемые/выделенные в нём ресурсы.
### Hyper-v
Состояние ВМ, на каком инстансе находится, потребляемые RAM, CPU.
### MSSQL 
список баз, на каком инстансе расположена. Размер и тип бд. 
### IIS
Список сайтов, их состояние, на каких портах опубликованы, путь к расположению. 
### ZABBIX 
От каких хостов zabbix получает данные с zabbix агентов
